基本ルール

対局形式

  • 4人打ち麻雀（東風戦 / 東南戦）
  • 初期持ち点: 25,000点
  • 和了には必ず1翻以上の役が必要（役なし和了禁止）

牌

  • 萬子(1m-9m)、筒子(1p-9p)、索子(1s-9s): 各4枚 = 108枚
  • 風牌(東南西北): 各4枚 = 16枚
  • 三元牌(白發中): 各4枚 = 12枚
  • 合計: 136枚（赤ドラなし）

王牌

  • 王牌14枚（嶺上牌4枚 + ドラ表示牌5枚 + 裏ドラ表示牌5枚）
  • ドラ表示牌: 初期1枚、槓ごとに1枚追加（最大5枚）
  • 裏ドラ: リーチ和了時のみ開示

局の進行

  • 親は東家（席番号0）から開始
  • 親が和了 → 連荘（本場+1）
  • 子が和了 → 親交代、本場リセット
  • 流局時（東場）→ 親がテンパイなら連荘、ノーテンなら親流れ
  • 流局時（南場）→ 親は常に続行（本場+1）

流局時のテンパイ料（ノーテン罰符）

  • 流局時、場に3,000点をノーテン者からテンパイ者へ移動
  • テンパイ1人 / ノーテン3人: テンパイ者+3,000 / ノーテン者各-1,000
  • テンパイ2人 / ノーテン2人: テンパイ者各+1,500 / ノーテン者各-1,500
  • テンパイ3人 / ノーテン1人: テンパイ者各+1,000 / ノーテン者-3,000
  • 全員テンパイ or 全員ノーテン: 点数移動なし

終了条件

  • 規定の場風が終了（東風戦: 東場まで / 東南戦: 南場まで）
  • 誰かの持ち点が0点未満（飛び）

鳴き（副露）

種類条件説明
チー上家の捨て牌順子を完成させる
ポン他家の捨て牌刻子を完成させる
大明槓他家の捨て牌手牌3枚+捨て牌で槓子
暗槓手牌のみ手牌4枚で槓子（門前維持）
加槓ポン済みポンした刻子に4枚目を加える

リーチ

  • 門前（副露なし）でテンパイ時に宣言可能
  • リーチ宣言時に1,000点の供託
  • リーチ後は手牌変更不可（ツモ切りのみ）
  • 和了時に裏ドラを開示

ドラ

種類説明実装
表ドラドラ表示牌の次の牌がドラ実装済み
裏ドラリーチ和了時に開示。裏ドラ表示牌の次の牌実装済み
槓ドラ槓するたびにドラ表示牌が1枚増加実装済み
赤ドラ赤牌（5m/5p/5sの赤版）未実装

ドラの計算方法

  • 数牌: 表示牌の次の数字（9→1に戻る）
  • 風牌: 東→南→西→北→東
  • 三元牌: 白→發→中→白
